Qual é a diferença entre linguagem de programação Server Side e Client Side?

Server Side

O código é executado no servidor, antes que a resposta seja enviada para o cliente (navegador do usuário).

Exemplos:

  • Python
  • Ruby
  • Java
  • PHP

Client Side

O código é executado no navegador do usuário, após a página ser carregada.

Exemplos:

  • HTML
  • CSS
  • JavaScript
  • Assembly
